Introdução:

O Eloquent ORM incluído no Laravel fornece uma implementação do ActiveRecord 
simples e bonita para trabalhar com seu banco de dados. Cada tabela de banco de dados 
tem um "Modelo" correspondente que é usado para interagir com essa tabela. 
Os modelos permitem consultar dados em suas tabelas, além de inserir novos registros na tabela.

Antes de começar, certifique-se de configurar uma conexão com o banco de dados. 
Para mais informações sobre como configurar seu banco de dados, confira a 
documentação .config/database.php

SITE documentação para ELOQUENT: https://laravel.com/docs/5.4/eloquent

comando para instalar:
> composer require illuminate/database:5.4.13
